# Job Khojo 💼

A full-stack job portal built using the MERN stack.

While preparing for internships and exploring different hiring platforms, I became curious about how job portals actually work behind the scenes. To understand the complete recruitment workflow, I built **Job Khojo**, where recruiters can post jobs and manage applicants while candidates can search, apply, and track opportunities.

## Live Demo

🚀 https://job-khoj-3.onrender.com/

## Features

⭐ Secure Authentication & Authorization using JWT

⭐ Recruiter and Candidate role-based access

⭐ Company profile creation and management

⭐ Job posting, updating, and deletion

⭐ Job search and filtering

⭐ Resume upload and management

⭐ Job application tracking

⭐ Applicant management dashboard

⭐ Protected routes and secure workflows

⭐ Responsive UI built with React and Tailwind CSS

## Tech Stack

* React JS
* Tailwind CSS
* Redux Toolkit
* Node JS
* Express JS
* MongoDB
* JWT
* Cloudinary
* Render

## Key Engineering Highlights

* Designed and implemented database relationships between Users, Companies, Jobs, and Applications
* Built role-based authentication and authorization for recruiters and candidates
* Integrated resume uploads using Cloudinary
* Developed complete job and application management workflows
* Managed global application state using Redux Toolkit
* Built and deployed a production-ready MERN stack application

## Application Workflow

### Candidate

* Create an account
* Upload resume
* Search jobs
* Apply for jobs
* Track applications

### Recruiter

* Create company profile
* Post and manage jobs
* View applicants
* Manage recruitment process

## Core Entities

* User
* Company
* Job
* Application

These entities work together to simulate a real-world hiring ecosystem between recruiters and job seekers.
